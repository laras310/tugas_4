# BarScan : Barcode Scanner

Aplikasi ini dapat memindai barcode dan mengarahkan pengguna ke link pada barcode.

## Package
Package yang digunakan:
- BarCode Scanner
- WebBrowser
- BlurView

## Installation
Aplikasi hanya untuk pengguna android. Install apk melalui link di bawah:
